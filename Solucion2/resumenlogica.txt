Liga deportiva
Actualmente, la liga presenta problemas en:
El control de jugadores habilitados para competir
# Por esto requieren listado de jugadores asegurados

La organización de equipos por categoría

La generación de listados confiables para control administrativo.
El sistema debe:
Gestionar equipos y jugadores (ABM)
Validar datos ingresados por el usuario
Generar listados según distitntos criterios

Funciones del programa basicas

#Equipos
Altas de equipos con asignacion automática de nombre
Baja de equipos
Modificaciones de equipos

#Jugadores
Alta de jugadores con validaciones segun restricciones
Baja de jugadores
Modificaciones de jugadores

#Funcionalidades
Listar jugadores asegurados
Listar jugadores: (se puede elegir una)
- Ordenados por edad
- Agrupados por categoria

#Otros reportes
-Jugador más joven y más viejo
-Promedio de edad general


ESTRUCTURAS
#JUGADORES
Deben tener
- DNI (único)
- Nombre
- Apellido
- Edad
- Equipo/s asignado/s 
- Seguro (Si/No)
- Afiliación (Si/No)

Se pediran estos datos como input y se crea la estructura de cada jugador, que podemos enlistar
Restricciones
- No se permiten DNIs duplicados
- Un jugador puede estar en más de un equipo dentro del mismo club siempre que la edad se lo permita

Tenemos entonces que
Revisar que el DNI no este repetido para alta de jugador y para que no haya dos jugadores en un mismo equipo



#CATEGORIAS
Listado de categorias y restricción de edad
- Infantiles (h/13)
- Cadetes (13 ~ 16) una vez cumplido 16 debe entrar a juveniles
- Juveniles (16~ 18)
- Primera (+18)
- Veteranos (+35)

#CLUB y EQUIPOS
- No se permiten DNIs duplicados
- La edad del jugador debe ser coherente con la categoría del equipo
- Un club puede tener múltiples equipos por categoría
los equipos se nombran automáticamente: Ejemplo: “Club Norte A”, “Club Norte B”, etc.
- Cada equipo pertenece a una única categoría
- Minimo de jugadores por equipo 9, excepto veteranos que puede tener 10
- El sistema debe poder detectar y reportar equipos que no cumplen con este requisito

Estructura de equipo
- Nombre del club
- Nombre de equipo (se designa autopmaticamente)
- Categoría única

AltaEquipo()

    │   ├── Si opcion == 0 → ingresar nombre nuevo club

    │   └── Si opcion > 0  → SeleccionarClub()
    │
    ├── SeleccionarCategoria()
    └── GenerarNombreEquipo()

BajaEquipo()
│
├── Si listaEquipos.Count == 0
│   └── "No hay equipos registrados", volver al menú
│
└── Si listaEquipos.Count > 0
    ├── SeleccionarEquipoPorClub() → nombreEquipoElegido
    │
    └── for listaEquipos
        └── Si NombreCompleto == nombreEquipoElegido
            ├── Si CantidadJugadores > 0
            │   └── "No se puede eliminar, tiene jugadores asignados"
            └── Si CantidadJugadores == 0
                ├── RemoveAt(i)
                ├── "Equipo eliminado exitosamente"
                └── break}

ModificarEquipo()
│
├── Si listaEquipos.Count == 0
│   └── "No hay equipos registrados", volver al menú
│
└── Si listaEquipos.Count > 0
    ├── SeleccionarEquipoPorClub() → nombreEquipoElegido
    │
    ├── Mostrar opciones:
    │   ├── 1. Eliminar un jugador del equipo
    │   └── 2. Eliminar todos los jugadores del equipo
    │
    ├── Si opcion == 1
    │   ├── for listaJugadores → armar jugadoresDelEquipo
    │   │   └── Si Equipos.Contains(nombreEquipoElegido)
    │   │       ├── jugadoresDelEquipo.Add(jugador)
    │   │       └── mostrar jugadoresDelEquipo.Count + Nombre + Apellido
    │   │
    │   ├── Si jugadoresDelEquipo.Count == 0
    │   │   └── "El equipo no tiene jugadores asignados"
    │   │
    │   └── Si jugadoresDelEquipo.Count > 0
    │       ├── SeleccionarOpcion(1, jugadoresDelEquipo.Count) → seleccion
    │       ├── jugadoresDelEquipo[seleccion - 1] → jugadorElegido
    │       ├── Confirmar()
    │       │   ├── Si confirma
    │       │   │   ├── QuitarEquipoDeJugador(jugadorElegido.DNI, nombreEquipoElegido)
    │       │   │   └── DecrementarJugadoresEquipo(nombreEquipoElegido)
    │       │   └── Si no confirma → "Modificación cancelada"
    │
    └── Si opcion == 2
        ├── Confirmar()
        │   ├── Si confirma
        │   │   ├── for listaJugadores
        │   │   │   └── Si Equipos.Contains(nombreEquipoElegido)
        │   │   │       └── QuitarEquipoDeJugador(dni, nombreEquipoElegido)
        │   │   └── CantidadJugadores = 0
        │   └── Si no confirma → "Modificación cancelada"

Confirmar()
├── do-while
│   ├── Console.Write("¿Confirma? (S/N): ")
│   ├── string input = Console.ReadLine().ToUpper()
│   └── Si input != "S" && input != "N"
│       └── "Error: ingrese S o N"
└── return input == "S"

│   └── Si fuera de rango → error

GenerarNombreEquipo(string nombreClub, Categoria categoria)
├── int cantidadExistentes = 0
├── for listaEquipos
│   └── si NombreClub == nombreClub Y Categoria == categoria
│       └── cantidadExistentes++
├── char sufijo = ('A' + cantidadExistentes)
│
└── return $"{nombreClub} {categoria} {sufijo}"
